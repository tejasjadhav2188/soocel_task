# Soocel Task React Login page
## Installation

Step 1: Clone the repository

```
 git clone https://github.com/tejasjadhav2188/soocel_task.git
```
 Step 2 : Change directory to soocel task
```
 cd soocel_task
```

Step 3 : Install Node Modules
```
 npm install
```

Step 4 : Run the app
```
 npm run dev
```
